## $(date +%Y-%m-%d) - Fixed horizontal overflow on ads-bar component on mobile breakpoints
**Learning:** Found that applying fixed `min-width` coupled with negative margins (`margin-left: -25px`) to create overlapping flex items inside a fluid container can cause severe horizontal overflow and break layout constraints on narrow screens when elements begin to wrap or squish.
**Action:** When designing overlapping flex items, prefer using standard CSS grid overlapping or flex distribution with `width: 100%` on the individual items under mobile media queries to ensure they break naturally instead of forcing rigid horizontal boundaries.
## $(date +%Y-%m-%d) - Prevented horizontal overflow caused by negative margins on ads-bar component
**Learning:** Found that applying negative horizontal margins (like `margin-left: -10px`) combined with forced positive margins (`margin-right: 10px`) on edge items inside a `justify-content: space-around` flex container breaks the responsive bounds and triggers horizontal scrolling on mobile devices when space is constrained.
**Action:** When designing responsive flex containers on mobile breakpoints, ensure outer margins on the first/last children do not push content beyond the container width. Remove negative margin overrides and rely on standard `width: 100%` and internal padding to manage spacing safely within viewport boundaries.
## $(date +%Y-%m-%d) - Replaced rigid fixed widths with fluid max-widths on responsive nested elements
**Learning:** Found that applying fixed pixel widths (e.g., `width: 280px` for `.filters-sidebar` or `width: 100px` for `.product-thumb`) prevents elements from correctly scaling down on smaller and intermediate viewports, breaking responsive grids and overflowing flex column layouts.
**Action:** Always replace strictly fixed `width` pixel properties on scalable responsive containers with a fluid constraint pattern like `width: 100%; max-width: [target_pixels]px;` to allow the element to stretch up to its intended size but shrink safely on mobile screens.
